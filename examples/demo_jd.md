# 演示岗位描述 (Demo JD)

---

## 岗位名称：高级Python开发工程师

### 工作地点：北京 | 部门：技术中台

---

## 岗位职责

1. 负责公司核心业务系统的后端设计与开发
2. 参与微服务架构设计和系统重构
3. 编写高质量代码，进行Code Review
4. 与前端、产品团队紧密协作
5. 线上问题排查与性能优化
6. 技术方案设计和文档编写

## 任职要求

1. **3-5年** Python开发经验，有大型项目经验优先
2. 精通 **Django** 或 **Flask** 框架，熟练使用 Django REST Framework
3. 熟悉 **MySQL**、**Redis**、消息队列
4. 有微服务架构设计和实践经验
5. 熟悉 **Docker**，了解 **Kubernetes** 优先
6. 了解 CI/CD 流程
7. 本科及以上学历，计算机相关专业
8. 良好的沟通能力和团队协作精神

## 加分项

- 有高并发系统设计经验
- 了解前端技术（React/Vue）
- 有开源项目贡献
- 有技术博客或技术分享经验
- 有 AWS/阿里云 使用经验
